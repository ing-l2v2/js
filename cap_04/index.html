4.1. Objeto console

En el objeto console se tienen los métodos principales

    console.log(`Lo que se quiere ver en consola`);
    console.log(console);
    console.log(window);
    console.log(document);
    console.debug(`Mensajes de depuración`);
    console.warn(`Mensajes warm alertas o avisos`);
    console.error(`Mensajes de error`);
    console.info(`Mensajes informativos, muy similares a los log`);

Limpiar la consola

    console.clear()

console.table() imprime objetos (y arreglos) como tablas con formato.
Ej. console.table(Object.entries(console).sort()); 

    console.table({
      Timestamp: new Date().getTime(),
      OS: navigator["platform"],
      Navegador: navigator["appCodeName"],
      Idioma: navigator["language"],
    });

console.group("Title") para crear un grupo colapsable, console.groupCollapsed("Title") para crear un grupo colapsado, admite titulo y finalizar con console.groupEnd()

Es posible aplicar estilos CSS básicos

    console.log(
      "%cTexto del mensaje!",
      "color: #f709bb; font-style: italic; text-decoration: underline; font-size: 2.5em;"
    );

console.time("tagDeDegugging") para iniciar el debugging de las instrucciones de código, termina con console.timeEnd("tagDeDebugging")

console.dir(objetos/arrays) para mostrar grandes objetos o arreglos en formato conveniente. Para imprimir xml o html se usa dirxml(), ej. console.dir(window); console.dir(document);

Referencia en la url https://programacionymas.com/blog/super-console-log-javascript

4.2. Objeto DATE

Para la creación de un objeto de tipo fecha se usa la palabra clave new 

    let fecha = new Date();

Se puede establecer una fecha mediante la cantidad de milisegundo desde la fecha Ene 1 1970 

    let fecha = new Date(100000000);

Si el parámetro es un String en el formato "YYYY/MM/DD" opcional puede contener la hora en el mismo string como "HH:MM:SS" 

    fecha = new Date("2024/03/27");
    fecha = new Date("2024/03/27 05:14:18");

Crea un fecha a partir de componentes numéricos

    fecha = new Date(y, m, d, h, min, s, ms)

Los meses se contabilizan desde el 0 (Enero) al 11 (Diciembre) y los días de la semana del 0 (domingo) al 6 (sábado) 

    fecha.getMonth()    fecha.getDay()

getDate() - retorna el número de día del calendario en el mes 1 hasta 28, 29, 30 o 31. 

   fecha.getDate()

getFullYear() - Retorna el año en 4 cifras

   fecha.getFullYear()

Para obtener la hora en formato 24, minutos, segundos y milisegundos se cuenta con: 

   fecha.getHours()    fecha.getMinutes()    fecha.getSeconds()    fecha.getMilliseconds()

fecha.toString() retorna fecha con formato "Fri Mar 01 2024 09:44:53 GMT-0500 (hora de Ecuador)" como Date().
fecha.toDateString() retorna la fecha con el formato "Fri Mar 01 2024".
fecha.toLocaleString() retorna la fecha con el formato "1/3/2024, 9:51:15" día/mes/año, hora.
fecha.toLocaleDateString() expresa la fecha con el formato "1/3/2024" dia/mes/año
fecha.toLocaleTimeString representa "9:54:53" hora:minutos:segundos
fecha.toTimeString imprime "09:55:59 GMT-0500 (hora de Ecuador)"

fecha.getTimezoneOffset() - retorna el valor 300 para Ecuador, al dividir para 60 se tiene 5 que es la zona horaria respecto a UTC hora de Londres.

Date.now() - retorna los segundos transcurridos desde "Ene 1 1970"

Fecha.getTime() - Devuelve el UNIX Timestamp: segundos transcurridos desde 1/1/1970.

Función para manejar formato de fecha recomendada

    function formatoFecha(fecha, formato) {
      const map = {
        dd:
          fecha.getDate() < 10
            ? "0" + fecha.getDate()
            : fecha.getDate(),
        mm:
          fecha.getMonth() < 9
            ? "0" + (fecha.getMonth() + 1)
            : fecha.getMonth() + 1,
        yy: fecha.getFullYear().toString().slice(-2),
        aaaa: fecha.getFullYear(),
      };

      return formato.replace(
        /dd|mm|yy|aaaa/gi,
        (matched) => map[matched]
      );
    }  
    formatoFecha(fecha, "dd-mm-aaaa");

4.3. Objeto Math

Es un objeto estático, siempre se invoca anteponiendo Math.

     console.table(Math);

Math.abs(number) - valor absoluto
Math.ceil(number) - redondear al entero posterior inmediato.
Math.floor(number) - redondear al entero anterior.
Math.round(number) - redondear al entero inmediato.
Math.sqrt(number) - raiz cuadrada de un número.
Math.pow(base, exponente) - base a la potencia
Math.sign(number) - -1 negativos, 1 positivos, 0 para 0
Math.random() - aleatorio entre 0 y 1

4.4. Operador de corto circuito

Cortocircuito OR - El valor de la izquierda en condicional siempre pueda validar a true, es el valor que se cargará por defecto. 

    variable = cadena || "Deconocido";
    variable = false || "Deconocido";

Cortocircuito AND - El valor de la izquierda en condicional siempre pueda validar a false, es el valor que se cargará por defecto. 

    variable = false && "Sin datos";
    variable = undefined && "Valor derecha";

Son secuencias de caracteres que forman un patrón de busqueda.
https://es.wikipedia.org/wiki/Expresi%C3%B3n_regular

Dos maneras de definir una expresión regular: usando función constructora RegExp la que requiere el patrón y las banderas (g = global todas las coincidencias, i = ignorar mayúsculas y minúsculas) o directamente entre dos barras. 

   let cadena = "lorem";
   let expreg = new RegExp("lorem", "g");
   let expreg2 = /lorem/;

4.7. Funciones anónimas autoejecutables

Fué un patrón de mucha importancia para la ejecución de código Inmediate Invoque Fuction Express, es una función que engloba todo el código que se requiere ejecutar. No tienen nombre. Usado en los patrones como jquery 


              (function(){
                console.log("Mi primer IIFE");
              })();
              (function(d, w, c){
                console.log("Mi primer IIFE");
                condole.log( d );
                condole.log( w );
                c.log( "Este es un console.log. Versión Clásica" );
              })(document, window, console);

              ((function (d, w, c) {
                c.log("Versión Crockford");
              })(document, window, console));

              +(function (d, w, c) {
                c.log("Versión Unaria");
              })(document, window, console);

              !(function (d, w, c) {
                c.log("Versión Facebook");
              })(document, window, console);

4.8. Módulos (import - export)

Módulos pueden invocarse desde otro .js

Los archivos js deben ser invocados al final, antes de la etiqueta /body, dado que el código js es bloqueante al invocar los .js se debe usar un nuevo valor al atributo type "module" 

    <script src="js/module.js" type="module"></script>

Para navegadores modernos se generaría un nomodule para que no tome en consideración el js 

    <script src="js/no-module.js" nomodule></script>

Para navegadores que no soporten +ES6, se generaría invocaciones a un js de una de las dos formas. 

    <script src="js/no-module.js"></script>
    <script src="js/no-module.js" nomodule></script>

Desde una archivo constantes.js para habilitar exportación solo de las funciones o constantes especificadas, puede existir una sola función exportada como default a la cual luego del export se le agrega la palabra default

La aplicación de export default para variables, constantes y funciones expresadas dado hoisting de JS deben declararse y luego exportarse como default, es decir, comportamiento diferente al de funciones declaradas, clases y objetos. 

    export const PI = Math.PI;
    let usuario = "no exportable";
    export let password = "puede exportarse";
    export default function saludar(){ console.log("Hola modulos ES6");}
    
    const hello = () => console.log("Funcion expresada para exportar como default, el default debe ser unitario");

    export default hello;

Para la importación desde un archivo modulos.js, luego de import las llaves y lo que se importa {PI, password} son necesarias porque es como que en un objeto recibiera el contenido importado, las import deben ser las primeras lineas del receptor, luego del from definir la ruta relativa al archivo origen. En el caso de una default debe definirse como primer elemento de recepción. La variable importado puede definirse con un alias con la palabra reservada as 

    import saludar, {PI, password} from "./constantes.js";
    import {aritmetica as calculo} from "./ aritmetica.js"

Se puede exportar objeto completo

    sumar(a, b){  return a + b;  }
    restar(a, b){  return a - b; }
    export const aritmetica = {
      sumar, restar
    }

Del lado de la importación sería

    import {aritmetica} FROM "./aritmetica.js";
    console.log(aritmetica.sumar(3, 4));
